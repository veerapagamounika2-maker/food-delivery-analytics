# 🍕 Food Delivery Analytics Dashboard

End-to-end analytics project analyzing food delivery
operations using Python, SQL Server, Power BI and Streamlit.

## 📊 Tech Stack

- Python, Pandas — Data cleaning
- SQL Server — Data analysis
- Power BI — Interactive dashboard
- Streamlit, Plotly — Web application

## 📈 Key Analysis

- Delivery time by weather conditions
- Traffic level impact on delivery
- Vehicle type performance
- Time of day patterns
- Courier experience analysis
- Distance vs delivery time
